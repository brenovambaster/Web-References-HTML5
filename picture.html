Tag HTML <picture>

Em HTML, as tags para imagens são <img> e <picture>. A tag <img> é usada para exibir uma única imagem, enquanto a tag <picture> é usada para imagens responsivas, permitindo que você especifique diferentes imagens (via tags source) que o navegador escolherá com base nas capacidades do dispositivo, como o tamanho da tela.

Exemplo

Uso da tag <picture>.

                                
                                <picture>
                                <source media="(min-width:650px)" srcset="img_pink_flowers.jpg">
                                <source media="(min-width:465px)" srcset="img_white_flower.jpg">
                                <img src="img_orange_flowers.jpg" alt="Flowers" style="width:auto;">
                                </picture>

Definição e uso

A <picture>tag dá aos desenvolvedores web mais flexibilidade na especificação de recursos de imagem. O uso mais comum do <picture> elemento será para direção de arte em designs responsivos. Em vez de ter uma imagem que é ampliada ou reduzida com base na largura da janela de visualização, várias imagens podem ser projetadas para preencher melhor a janela de visualização do navegador.

O elemento <picture> contém duas tags: uma ou mais tags <source> e uma tag <img> .

O navegador procurará o primeiro elemento <source> onde a consulta de mídia corresponde à largura da janela de visualização atual e, em seguida, exibirá a imagem correta (especificada no atributo srcset). O elemento <img> é necessário como o último filho do elemento <picture>, como uma opção de fallback caso nenhuma das tags de origem corresponda.
