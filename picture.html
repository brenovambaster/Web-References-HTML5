Tag HTML <picture>

Em HTML, as tags para imagens são <img> e <picture>. A tag <img> é usada para exibir uma única imagem, enquanto a tag <picture> é usada para imagens responsivas, permitindo que você especifique diferentes imagens (via tags source) que o navegador escolherá com base nas capacidades do dispositivo, como o tamanho da tela.
