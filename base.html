Tag <base>

Definição

A tag <base> define a URL base para todos os links e recursos relativos de um documento HTML. Ela deve ser utilizada dentro do elemento <head> e é útil quando se deseja centralizar os caminhos relativos, facilitando a manutenção dos links e recursos.

Exemplo de Uso

No exemplo abaixo, definimos a URL base para que todos os links ou imagens que usarem caminhos relativos sejam referenciados a partir de "https://www.exemplo.com/".

<!DOCTYPE html>
<html lang="pt-br">
    <head>
        <base href="https://www.exemplo.com/" target="_blank">
        <meta charset="UTF-8">
        <title>Documento com Base</title>
    </head>
    <body>
        <a href="pagina.html">Página Relativa</a>
    </body>
</html>

O link acima apontará para "https://www.exemplo.com/pagina.html" e será aberto em uma nova aba.

Atributos da tag <base>

Atributo Valor Descrição
href URL Define a URL base para todos os links e recursos relativos.
target _self, _blank, _parent, _top Define o destino padrão para a abertura de links e formulários.
