Tag HTML <sup>

Definição e uso

A tag <sup> é utilizada para definir texto sobrescrito, ou seja, texto exibido um pouco acima da linha padrão e com tamanho menor.

É comumente usada em expressões matemáticas, químicas, referências e notas.

Atributos

A tag <sup> não possui atributos específicos, mas aceita todos os atributos globais do HTML, como class, id e style.

Exemplo com estilo CSS

<style>
  sup { color: blue; font-size: 0.8em; }
</style>
