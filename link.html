Tag HTML <link>

Exemplo

Usada para vincular arquivos externos, como folhas de estilo CSS.

<head>
  <link rel="stylesheet" href="styles.css">
</head>

Definição e uso

A tag <link> define uma relação entre o documento HTML atual e um recurso externo. É usada principalmente para conectar folhas de estilo (CSS), ícones e fontes.

Essa tag deve aparecer sempre dentro da seção <head> do documento e não possui tag de fechamento.

Atributos

Atributo Valor Descrição
href URL Especifica o caminho para o recurso vinculado.
rel stylesheet / icon / alternate / preconnect / manifest Indica o tipo de relação entre o documento e o recurso.
type media_type Define o tipo de conteúdo do recurso vinculado.
media media_query Especifica para quais dispositivos ou condições o recurso é aplicado.
sizes 16x16 / 32x32 / etc. Usado com ícones para definir seus tamanhos.

Exemplo com ícone de site

<link rel="icon" href="favicon.ico" type="image/x-icon">
