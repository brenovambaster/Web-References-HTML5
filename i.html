Tag HTML <i>

A tag usada para destacar um texto com ênfase sem alterar seu significado semântico.

Exemplo

No exemplo abaixo, o texto dentro da tag <i> é exibido em itálico:

<p>A palavra <i>carpe diem</i> vem do latim e significa "aproveite o dia".</p>

Definição e uso

A tag <i> representa uma parte do texto que é destacada de maneira alternativa — normalmente em itálico — sem adicionar ênfase semântica, apenas estilística.

É frequentemente usada para termos estrangeiros, pensamentos internos, nomes científicos, ou palavras que precisam se diferenciar do restante do texto sem ganhar destaque semântico (como <em> faz).

Atributos

Atributo Valor Descrição
class Nome de classe Permite aplicar estilos CSS personalizados à tag.
id Identificador único Permite identificar o elemento unicamente dentro da página.
style Declaração CSS Permite adicionar estilos inline diretamente no elemento.

Boas Práticas e Observações

  • Semântica: Use <i> apenas quando o texto tiver uma aparência diferenciada, mas não um significado diferente. Para dar ênfase real, use <em>.
  • Estilo CSS: O itálico pode ser alterado via CSS, usando font-style: italic;.
  • Acessibilidade: Evite usar <i> apenas para destacar algo importante; leitores de tela não reconhecem essa tag como ênfase.
  • Uso comum: Palavras estrangeiras, pensamentos, termos técnicos, citações curtas e nomes científicos.

Outros exemplos

Veja alguns usos práticos da tag <i>:

Exemplo 1: Termo estrangeiro

<p>A expressão <i>déjà vu</i> é de origem francesa.</p>

Output

A expressão déjà vu é de origem francesa.

Exemplo 2: Pensamento interno

<p>Ela pensou: <i>Será que ele vai lembrar?</i></p>

Output

Ela pensou: Será que ele vai lembrar?
