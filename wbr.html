Tag HTML <wbr>

Exemplo

Usada para indicar uma possível quebra de linha em palavras longas.

<p>EstaPalavraExtremamenteLongaPodeSerQuebrada.</p>

Definição e uso

A tag <wbr> (Word Break Opportunity) indica onde o navegador pode quebrar uma palavra longa se necessário, evitando que o texto ultrapasse os limites do contêiner.

Ela não insere uma quebra de linha obrigatória, apenas marca um ponto onde a quebra é permitida se o espaço for insuficiente.

Atributos

A tag <wbr> não possui atributos específicos, mas pode usar os atributos globais do HTML.

Exemplo prático

<div style="width: 100px; border: 1px solid #ccc;">

</div>
